Atividades de JavaScript

A vendinha é um mini mercado virtual onde você pode escolher entre diferentes produtos. Nesta atividade, utilizamos a função alert() para exibir mensagens de boas-vindas e escolhas, e a função prompt() para capturar a escolha do usuário. Dependendo da escolha, um switch determina a mensagem a ser exibida.

A verificação de Idade é um sistema que controla o acesso ao site com base na idade do usuário. A função alert() é usada para saudar o usuário e solicitar a idade através de prompt(). Um if-else verifica se a idade é maior ou igual a 18 para permitir o acesso, caso contrário, o usuário é redirecionado usando window.location.

A Calculadora é uma ferramenta simples para realizar operações matemáticas básicas. Utilizamos document.getElementById para capturar os valores de entrada e o operador. Um switch realiza o cálculo com base no operador selecionado e innerText exibe o resultado.

O verificador de nota é uma aplicação que avalia se a nota do usuário está acima da média. A função alert() introduz a atividade e prompt() captura a nota. Um if-else verifica se a nota é maior ou igual a 7 para exibir uma mensagem de aprovação ou reprovação, e redireciona o usuário se reprovado.

O Dias da semana é um verificador que informa o dia da semana com base em um número de 1 a 7. Utiliza prompt() para capturar o número e um switch para mapear o número ao dia da semana correspondente, exibindo a resposta com alert().
